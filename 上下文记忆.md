## 环境依赖清单

### 后端环境 (Python)
- **Python 3.8+**
- 安装: `cd backend && pip install -r requirements.txt`
- 启动: `python run_simple.py`

### 前端环境 (uni-app)
- **HBuilderX**: https://www.dcloud.io/hbuilderx.html
- 打开 miniapp/ 文件夹即可运行

### 数据库（完整版需要）
- PostgreSQL 14+
- Redis 7+

## 数据源
- **股票列表**: 新浪财经API（每页80条，5511只A股）
- **实时行情**: 同花顺API（d.10jqka.com.cn）

## 启动方式
1. 双击 `启动股票行情.bat` 一键启动
2. 或 `cd backend && python run_simple.py`
3. 访问 http://localhost:8000

## 注意事项
1. 缓存TTL=300秒，首次加载约45秒
2. 给朋友分享用你的内网IP
3. 换设备开发需重新安装Python依赖
