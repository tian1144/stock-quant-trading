### 选股引擎（5阶段）
1. **硬性过滤**：剔除ST、负面新闻、停牌、涨跌停异常
2. **量价因子评分**（25%）：量比、换手率趋势、量价配合度、成交额、成交量稳定性
3. **资金因子评分**（30%）：主力净流入、大单占比、连续净流入天数、近3日趋势
4. **技术因子评分**（30%）：均线多头、MACD金叉、KDJ超卖回升、RSI、布林带、支撑位
5. **综合评分**：加权总分排序取Top50

### 反追涨杀跌设计
**买入反追高规则：**
- 日内涨幅>3%禁止买入
- 5分钟急涨>1.5%禁止买入
- 涨>1%且未从高点回落1%禁止
- 连涨5日禁止买入
- 价格偏离VWAP>3%禁止买入

**卖出反杀跌规则：**
- 止损触发才卖出（不因恐慌随意卖）
- 单日跌>5%不恐慌卖出（除非止损）
- 移动止盈：盈利后从峰值回撤5%才卖
- 技术卖出需MACD死叉+量缩+破MA5多条件确认

### 风控配置
```python
RISK_CONFIG = {
    "initial_cash": 200000.0,           # 初始资金20万
    "max_single_position_pct": 0.20,    # 单票最大20%（4万）
    "max_total_position_pct": 0.80,     # 总仓位最大80%（16万）
    "max_position_count": 10,           # 最多持有10只
    "default_stop_loss_pct": 0.05,      # 5%止损
    "default_take_profit_pct": 0.10,    # 10%止盈
    "max_daily_loss_pct": 0.03,         # 日亏损3%暂停交易
    "max_single_trade_amount": 40000,   # 单笔最大4万
    "commission_rate": 0.00025,         # 佣金万2.5
    "stamp_tax_rate": 0.0005,           # 印花税万5（卖出）
    "cooldown_after_sell_minutes": 30,  # 卖出后30分钟冷却
    "max_daily_trades": 20,             # 每日最多20笔
}
```

### 定时任务（APScheduler）

- 每15分钟：新闻刷新
- 每10分钟：候选股资金流向更新
